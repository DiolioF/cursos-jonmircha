Etiquetas semanticas Estructurales

Div - Es una etiqueta contenedora que no tiene semantica
header - Cabecera de un sitio web o una seccion
Main- Define la seccion princiapl del docuemnto, sólo puede existir una etiqueta main por documento
Footer - Pie de pagina de un sitio web o de una seccion
nav - Representa una navegacion
Article - Representa una seccion de autocontenido (que por si sola se explica)
aside - Representa contenido complementario o secundario
section - Representa una seccion de conteido generico
Address - Representa una informacion de contacto

Etiquetas de bloque vs etiquetas de linea

Div - Es la etiqueta en bloque por excelencia, Las etiquetas en bloque son aquellas que generan saltos de linea con sus etiquetas hermanas
Span - Es la etiqueta de linea por excelencia, Las etiqeutas de linea solo ocupan el espacio necesario que tiene su contenido ☝️Hacia Arriba

Estilos CSS en HTML

Hay 3 formas de agregar estilo CSS:

  • 1° - En la etiqueta HTML con el atributo Style="valor css"
  • 2° - En la cabezera abriendo una etiqueta Style
  • 3° - Desde un archivo externo CSS, linqueandolo en el header con la etiqueta link

Javascript en HTML

Hay 2 formas:

  • 1° - Escribiendo codigo javascript dentro de la etiqeuta Script
  • 2° - Desde un archivo javascript externo, enlazandola con la etiqueta script + el atributo src="ruta"
